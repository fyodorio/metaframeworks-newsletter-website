## The Bad

So who's faster than metaframeworks vendors in coming up with new surprises and spicy topics? Of course it is the hackers. Some of them do it for good, and some bear cruel intentions.

On the ~~good~~ ethical side, Astro continues to be a target for stress-testing by the metaframeworks hacker Rachid Allam and this time [the high-criticality vulnerability culprit is Astro's server islands](https://zhero-web-sec.github.io/research-and-things/unlocking-reflected-xss-in-the-astro-framework) providing a pathway for notorious cross-site scripting (XSS). Server islands, as a metaphor and a feature, have not been lucky lately in general. [Another (_critical!_) vulnerability](https://react.dev/blog/2025/12/03/critical-security-vulnerability-in-react-server-components) was found by Lachlan Davidson in React Server Components (RSC) and it can lead to an even more dangerous remote code execution (RCE) flaw. Maybe it's time to think about the aftermath growing metaframeworks complexity leads us to. Anyways, not until we unpack the Santa's deliveries this year, so for now make sure to update to the latest non-vulnerable versions.

On a more evil side, the [only boring `npm` hacking attack that got its own cool name ‚Äî Shai Hulud ‚Äî is back](https://socket.dev/blog/shai-hulud-strikes-again-v2), causing drastic damage to the `npm` ecosystem and lots of popular packages, and leading to some [interesting and thoughtful post-mortems](https://posthog.com/blog/nov-24-shai-hulud-attack-post-mortem) from the vendor community this time. Some less loud [vulnerabilities accompany it too](https://socket.dev/blog/north-korea-contagious-interview-npm-attacks), which again emphasizes that even though the tooling providers try to strike back, it is not enough and we need to think better.

But for now the companies working on tooling think more about other topics, like finding more sustainable ways of doing their work. On this wave, Bun decided to join the AI army and [declared the acquisition](https://bun.com/blog/bun-joins-anthropic) by ~~Vercel~~ (üòÆ‚Äçüí®) Anthropic. Is it good or bad? Hard to say now, but I really like Bun's way of doing their job and their thought leadership strategy. Obviously I'm not the only one (_congrats Anthropic!_) and I want to believe that it is for the better in this industry and at this time.
